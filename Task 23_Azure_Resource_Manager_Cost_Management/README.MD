# Task 23 — Azure Resource Manager & Cost Management

## 📌 Objective

To understand Azure Resource Manager (ARM), resource grouping, and cost tracking using Azure Cost Management.

---

## 🏗️ Architecture Overview

Azure Subscription  
→ Resource Group (`susovan-azure-t18-t24-rg`)  
→ Multiple Azure Resources  
→ Cost Management & Billing  

This demonstrates centralized deployment, organization, and financial monitoring within Azure.

---

## 🛠️ Services Used

- Microsoft Azure
- Azure Resource Manager (ARM)
- Resource Groups
- Azure Cost Management + Billing

---

## 🚀 Steps Performed

### 1️⃣ Resource Group Management

- Created Resource Group:
  ```
  susovan-azure-t18-t24-rg
  ```
- Deployed multiple resources inside the group (VM, Storage, Networking, etc.).
- Verified logical grouping and centralized management.
- Confirmed resources can be managed collectively.

---

### 2️⃣ Cost Analysis

- Navigated to:
  ```
  Cost Management + Billing
  ```
- Opened **Cost Analysis**.
- Viewed:
  - Total subscription spending
  - Service-level cost breakdown
  - Daily and monthly trends
- Adjusted filters for time range and resource group.

---

### 3️⃣ Budget Creation (Optional)

- Created budget:
  ```
  $10
  ```
- Configured alert threshold:
  - 80% usage notification
- Verified alert configuration for proactive cost control.

---

## 🔑 Key Concepts

- Azure Resource Manager (ARM) manages deployment and lifecycle of resources.
- Resource Groups logically organize related resources.
- Cost Management provides visibility into spending.
- Budgets help control and monitor cloud expenses.
- Centralized management improves governance and accountability.
